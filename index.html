Insight Data Science Program

Currently, I am a fellow at the Insight Data Science Program. For my Insight project, I consulted with a subscription e-commerce startup to develop a forecasting model for the demand of their products. The forecasting algorithm takes as input customer transaction data, models the seasonality, growth, churn, and probability of demand for two different types of customers, and provides predictions for the demand of 24 products up to two months in the future. The mean weighted error was estimated at 6.7% using a training set from August 2014 to August 2015 and a testing set from September 2015. The dataset is stored in a PostgreSQL database and for the analysis I used Python, Pandas, NumPy and SciPy.

Research

My research area is Solar Physics. I use machine learning methods to predict solar storms, such as flares and coronal mass ejections. This is an unbalanced classification problem since solar eruptions are rare events and only a small percentage of the dataset contains observations of the positive class. My work involves developing Python code for data cleaning, feature extraction, and feature engineering, implementation of feature selection and appropriate validation methods for unbalanced datasets, and optimization and hyperparameter tuning of machine learning algorithms. Most of my work is done with supervised learning algorithms such as Support Vector Machines, Random Forests, AdaBoost and Gradient Boosting, and Neural Networks. I presented the results of my work in August 2015 at the XXIX International Astronomical Union (IAU) General Assembly in Hawaii, which was the largest astronomy conference ever held.

As a PhD student, I used helioseismic methods to investigate the interior of the Sun. I designed and implemented noise reduction methods based on computation of billions of cross correlations, and detected for first time emerging sunspots beneath the solar surface. The results of this work were published in Science magazine and generated media coverage in over 50 media outlets, including NASA Science News. For more information about this project, please visit the page of the Helioseismic and Magnetic Imager.

Side Projects

Last summer I participated at the Kaggle Caterpillar Tube Pricing competition. The goal of this project is to develop a machine learning model and predict the price of industrial tube assemblies. The dataset is comprised of 21 relational tables. I merged all the tables into one large dataframe and I used imputation techniques to replace missing values. I also used advanced feature engineering methods to create new features with high predictive value. Then, I implemented feature selection methods based on feature correlations and feature importances (computed with a Random Forest Regressor) to reduce the number of features from 300 to about 60. Finally, I computed a blend of Gradient Boosting models using the XGBoost library and Bagging. My best submission achieved a score of about 0.2125 on the private leaderboard and was ranked in the top 3% among more than 1300 teams. A more detailed summary of my approach as well as my code are posted in my Github page.

Recently, I also developed a customized recommendation system for traveling. This application takes as input the user's preferences for airfare, weather, safety, dining and nightlife, and recommends travel destinations using Yelp, QPX, and Weather Underground APIs as well as city crime data. A more detailed description of the application as well as the Python code are posted in my Github page.
